# SoftGrowTech_LibrarySystem
📌 Description

This project is a **Library Management System** developed in C++ that manages books, users, and transactions efficiently. It allows adding and displaying books, registering users, and handling book issue and return operations.

The system uses file handling to store data persistently and is designed using a menu-driven approach for ease of use. It demonstrates core programming concepts including file handling, structured programming, and basic system design in C++.
